# Physical Mac Setup

Use these scripts when setting up OpenClaw on a **physical Mac** (MacBook, Mac Mini, iMac, Mac Studio, etc.).

## Files

| Order | File | Description |
|-------|------|-------------|
| 1 | [01-install-openclaw.md](01-install-openclaw.md) | Install Homebrew, Node.js, and OpenClaw |
| 2 | [02-imessage-setup.md](02-imessage-setup.md) | Configure iMessage integration (optional) |
| 3 | [03-reference.md](03-reference.md) | Useful paths and troubleshooting |

## Quick Start

1. Start with **01-install-openclaw.md** to install the basics
2. If you want iMessage integration, follow **02-imessage-setup.md**
3. Refer to **03-reference.md** for paths and troubleshooting
