# OpenClaw Setup Scripts

Scripts for setting up OpenClaw on a fresh Mac.

## Choose Your Setup

| Folder | Use When |
|--------|----------|
| [Physical-Mac-Setup](Physical-Mac-Setup/) | Setting up a MacBook, Mac Mini, iMac, Mac Studio, etc. Includes iMessage integration. |
| [Virtual-Machine-Setup](Virtual-Machine-Setup/) | Setting up a cloud VM, local VM, or automated provisioning. No iMessage. |

## Not Sure Which to Use?

- **Physical Mac** = A real Mac you can touch. Use `Physical-Mac-Setup/`.
- **Virtual Machine** = A Mac running inside another computer (AWS, Azure, Parallels, etc.). Use `Virtual-Machine-Setup/`.
